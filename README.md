# Media App

A reactive native app to stream media contents from django server.


### Requirements
```
- nodejs v16
- andorid sdk with env path
```


### Setup 
```
- git pull 
- npx install
- npx expo start
npx expo start --web
```

### Run 

> npx expo run:android
> npx react-native run-android

### Extra 

> npx clear-npx-cache

> cd android/ && gradlew clean
